# Figure Contract

Before plotting, identify:

1. The claim the figure supports.
2. The data source and units.
3. The comparison baseline or physical reference.
4. The output format: SVG or PDF for vectors, PNG/TIFF for raster exports.
5. The backend: Python or R.

For quantitative manuscript figures, write `figure_data_map.json` and record
source files, filters, groups, metrics, transformations, displayed values,
dual-axis justification, colorblind-safe palette status, uncertainty visibility,
subset selection, and low-support subgroup warnings.
