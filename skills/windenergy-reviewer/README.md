# `windenergy-reviewer` skill

`windenergy-reviewer` simulates a conservative pre-submission review package for
wind, renewable-energy, power-system, and AI-for-energy manuscripts.

It is designed to complement `windenergy-submission` and `windenergy-orchestrator`.
It identifies rejection risks before submission and returns three reviewer
reports, a cross-review synthesis, and an author action list.

Install the complete folder, including `manifest.yaml`, `static/`,
`references/`, `evals/`, and `agents/`.
