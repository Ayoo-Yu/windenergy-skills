# Difficult Cases

Use this file when a reviewer request cannot be fully satisfied.

## New Data or Experiment Not Available

Be transparent. State what was added instead, why the requested work is outside
the current revision, and what limitation was added.

## Conflicting Reviewer Requests

Respect both concerns, explain the compromise, and identify where the manuscript
was changed.

## Unsupported Claim

Soften or remove the claim. Do not defend a statement that the manuscript cannot
support.

## Out-of-Scope Request

Explain why the request exceeds the study scope and add a limitation or future
work statement if appropriate.
