# Chart Types

| Claim | Recommended chart |
|---|---|
| Forecast tracks actual power | Time series with forecast, actual, and error band |
| Model improves accuracy | Metric bar/point plot with confidence interval |
| Error depends on wind regime | Box/violin plot by wind-speed bin or operating region |
| Model handles ramp events | Event-window line plot with ramp annotations |
| Physical consistency | Power curve scatter with theoretical/empirical curve |
| Directional resource pattern | Wind rose or polar histogram |
| Module contribution | Ablation heatmap or grouped metric table |
| Spatial dependency | Farm layout, wake graph, or adjacency heatmap |

Avoid chart types that hide sample size, uncertainty, or operating-regime
boundaries when those factors are central to the claim.
