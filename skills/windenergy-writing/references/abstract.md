# Abstract Writing Guide for Renewable Energy Papers

Load `../_shared/core/narrative-principles.md` with this guide.

## Goal

Write an abstract that is accurate, memorable, and evidence calibrated.

## Pre-Writing Questions

1. What field-level tension, unresolved problem, or wrong intuition motivates
   the paper?
2. What does the manuscript reveal: method, mechanism, boundary, design
   principle, benchmark result, or decision rule?
3. What evidence is strong enough to keep in the abstract?
4. What is the take-home message a reader should remember?

## High-Impact Abstract Pattern

Use this pattern for research articles and foreground submission text.

1. Field problem and tension.
2. Gap or wrong intuition.
3. What the paper does.
4. Compressed evidence, with only the numbers that anchor the main claim.
5. Implication and take-home message.

Technical details and experiment scale should serve the main judgment. Do not
start with run counts, model names, or implementation details unless they are
the contribution.

## Generic Skeleton

```text
[Energy system task] increasingly depends on [decision or physical setting],
yet [current limitation or unresolved assumption] still makes [outcome] hard to
trust. This study examines [question] by [approach]. The results show that
[core finding], with [one or two key evidence anchors]. The finding changes how
[method, asset, or decision] should be selected: [take-home message].
```

## Multiple-Contribution Skeleton

```text
This paper addresses [energy problem] where [field tension]. We make three
contributions. First, [contribution and why it matters]. Second, [contribution
and evidence type]. Third, [boundary, design rule, or deployment implication].
Across [brief evidence scope], the results support [main judgment]. The
take-home message is [decision principle or field update].
```

## Quality Checklist

1. Does the opening create a real problem rather than generic background?
2. Is the core finding visible before secondary numbers?
3. Are technical details compressed?
4. Does the final sentence state a take-home message?
5. Is claim strength unchanged from the evidence chain?
