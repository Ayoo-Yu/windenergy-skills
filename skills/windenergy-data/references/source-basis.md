# Source Basis

Use this file to keep data-availability work grounded in target-journal policy
and actual manuscript facts.

## Source Hierarchy

1. Current target-journal guide for authors and submission system prompts.
2. Elsevier research-data and data-statement guidance for energy journals.
3. Repository-specific instructions.
4. Author-provided data rights, licenses, and operator agreements.

## Working Implications

- Current journal instructions override this local file.
- Public data should be cited or linked precisely.
- Restricted data must include a clear reason and access route if any.
- Data and code statements must not promise resources that do not exist.
- If generative AI or automated tools affected data analysis, route disclosure
  questions to `windenergy-submission`.
