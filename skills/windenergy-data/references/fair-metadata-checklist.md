# FAIR Metadata Checklist

## Findable

- DOI or stable URL.
- Descriptive title with asset, site/region, time span, and sampling interval.
- Energy keywords such as wind power, SCADA, NWP, ERA5, forecast, turbine.

## Accessible

- Clear repository landing page.
- License or access conditions.
- Private reviewer link if the record is embargoed during review.

## Interoperable

- CSV, NetCDF, Parquet, HDF5, or another documented open format.
- SI units, time zone, coordinate system, and missing-value codes.
- Data dictionary for each variable.

## Reusable

- Versioned release that matches the manuscript.
- README describing cleaning, filtering, curtailment handling, and train/test
  splits.
- Scripts or notebooks that reproduce figures or tables when possible.
