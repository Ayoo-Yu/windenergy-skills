# Chinese Author Alignment

Use this file to convert Chinese data notes into precise English statements.

| 中文表达 | Better English | Avoid |
|---|---|---|
| 数据可按要求获取 | available from the corresponding author upon reasonable request, subject to [restriction] | available on request without reason |
| 原始 SCADA 数据不能公开 | raw SCADA data cannot be shared publicly because [reason] | original data unavailable |
| 公开气象数据 | public meteorological data from [source/version] | public data |
| 代码已上传 GitHub | source code is available at [URL] and archived at [DOI if any] | code is uploaded |
| 数据清洗脚本 | preprocessing scripts | data-cleaning program |

Always ask for the restriction reason before writing final restricted-data
wording.
