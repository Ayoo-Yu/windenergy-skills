# Ethics and Source Policy

Use these rules across the renewable Skill suite.

## Evidence Integrity

- Do not fabricate citations, DOIs, author details, affiliations, grants, reviewer suggestions, datasets, code repositories, model results, or figure content.
- Do not strengthen claims beyond supplied evidence.
- Do not cite generic AI literature as support for wind-energy, power-system, or renewable-energy claims unless the cited paper directly supports that claim.
- Treat operator SCADA data, grid data, private reports, and proprietary code as restricted until the user states otherwise.

## Submission Integrity

- Journal instructions can change. For final submission checks, verify the current official English guide when web access is available.
- Use local `期刊整理` files as a read-only source archive. If local files conflict with current official guidance, follow the official guidance and report the conflict.
- Do not write a final generative AI declaration until the user states which AI tools were used and for what purpose.
- Do not list AI tools as authors.

## Search Policy

- Use English-language official, publisher, DOI, CrossRef, OpenAlex, Semantic Scholar, arXiv, standards-body, dataset, or institutional sources.
- Do not use Chinese websites as search sources.
- Label preprints, reports, and non-peer-reviewed sources clearly.
