# Intake and Routing

| Mode | Trigger | Output |
|---|---|---|
| `draft` | reviewer comments plus author notes | Point-by-point response draft |
| `audit` | existing response draft | Risk report and missing actions |
| `revise` | user wants response text improved | Revised response text |
| `map` | comments plus manuscript changes | Comment-to-change matrix |
| `council-review` | reviewer comments need multi-expert revision strategy | Consensus plan and response draft |

Ask for the revised manuscript only when line numbers, section locations, or
exact wording are required and not provided.
