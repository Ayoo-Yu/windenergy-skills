# Keywords Section Fragment

## Default Requirements

- Use 6 keywords by default for `common-18`
- Use target profile limits when a journal is named
- Written in **English**
- Avoid multi-word keywords using "and" or "of"
- Abbreviations only if firmly established (e.g., AI, ML, SCADA)
- Used for indexing purposes

## Keyword selection strategy

### Step 1. Identify core concepts

From the manuscript title, abstract, and main claims, extract:

1. The renewable energy technology (e.g., wind power, solar PV)
2. The application domain (e.g., forecasting, optimization, monitoring)
3. The method family (e.g., deep learning, Transformer, CFD)
4. The specific technique if novel (e.g., physics-informed, attention mechanism)

### Step 2. Prioritize discoverability

Keywords should maximize the paper's discoverability via search engines and indexing services. Prefer terms that researchers in the field would actually search for.

### Step 3. Avoid redundancy with title

If the title already contains key terms, you may still include them as keywords (different search systems index title and keywords separately), but prioritize adding terms NOT in the title.

```text
Wind resource assessment; Weibull distribution; Wind atlas;
ERA5; Capacity factor; Spatial analysis
```

## Format

Present keywords separated by semicolons:

```text
Keywords: [keyword 1]; [keyword 2]; [keyword 3]; [keyword 4]; [keyword 5]
```

## QA checklist

- [ ] Keyword count matches the target profile
- [ ] All in English
- [ ] No multi-word phrases with "and" or "of"
- [ ] Abbreviations only if established
- [ ] Not duplicating the entire title
- [ ] Maximizing discoverability
