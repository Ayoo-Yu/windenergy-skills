# Action Mapping

| Action | Use when |
|---|---|
| `TEXT_REVISED` | Wording, structure, or explanation was changed |
| `ANALYSIS_ADDED` | New metric, ablation, robustness check, or table was added |
| `FIGURE_REVISED` | Figure, caption, axis, or panel was changed |
| `CITATION_ADDED` | Relevant literature was added or replaced |
| `CLAIM_SOFTENED` | Claim strength was reduced to match evidence |
| `LIMITATION_ADDED` | Boundary condition or missing validation was acknowledged |
| `NO_CHANGE_JUSTIFIED` | Authors respectfully disagree with evidence |
| `AUTHOR_INPUT_NEEDED` | The response cannot be completed from available facts |

Every final response should include at least one action or a clear justification.
