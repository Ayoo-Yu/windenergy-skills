# Output Contract

Default response:

1. `Review setup`
2. `Reviewer 1 - Energy-system significance`
3. `Reviewer 2 - Method and reproducibility`
4. `Reviewer 3 - AI/statistical validity`
5. `Cross-review synthesis`
6. `Author action list`
7. `Unsupported or missing evidence`

Each finding must include severity, affected manuscript part, evidence basis,
and concrete revision action. Use `AUTHOR_INPUT_NEEDED` when the author must
provide data, line numbers, figures, citations, or journal choice.
