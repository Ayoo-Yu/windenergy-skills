# Workflow

1. Inventory the manuscript package and target journal.
2. Identify paper type, topic, claimed contribution, main evidence, and missing
   material.
3. Run three reviewer lenses: energy significance, method reproducibility, and
   AI/statistical validity.
4. Separate technical soundness from journal-fit risk.
5. Build a consensus table with severity, evidence basis, and required author
   action.
6. Return reviewer reports plus an action list that can feed
   `windenergy-orchestrator`, `windenergy-response`, or `windenergy-polishing`.
