# Statement Patterns

## Public Data and Code

```text
The processed data supporting the findings of this study are available from
[Repository] at [DOI]. Source code is available at [GitHub URL] and archived at
[Zenodo DOI] under the [license] license.
```

## Restricted SCADA Data

```text
The wind farm SCADA data used in this study were provided by [operator] under a
data-use agreement and cannot be shared publicly because they contain
commercially sensitive operational information. Aggregated statistics needed to
interpret the results are provided in [supplement/table]. Public meteorological
data were obtained from [source] at [URL/DOI].
```

## Reused Public Data

```text
[Dataset name] data are publicly available from [provider/repository] at
[URL/DOI]. The version/access date used in this study was [version/date].
```

## No New Data

```text
No new datasets were generated in this study. The analysis used publicly
available data from [sources], as described above.
```

## Author Action Flags

Use `AUTHOR_INPUT_NEEDED` when any of these are missing:

- operator name or permission wording
- repository DOI or private reviewer link
- data license
- access date/version
- exact code repository
