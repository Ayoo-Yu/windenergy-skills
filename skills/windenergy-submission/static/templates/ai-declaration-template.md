# AI Declaration Template

## When AI tools were used

Place this section according to the target journal profile, commonly at the end
of the manuscript before the References.

```text
Declaration of generative AI and AI-assisted technologies in the
manuscript preparation process

During the preparation of this work the author(s) used [TOOL NAME AND
VERSION] in order to [SPECIFIC PURPOSE]. After using this tool/service,
the author(s) reviewed and edited the content as needed and take(s) full
responsibility for the content of the published article.
```

## Common tool/purpose fill-ins

| Tool | Purpose example |
|------|----------------|
| Claude (Anthropic) | improve language clarity and readability |
| ChatGPT (OpenAI, GPT-4) | organize literature review notes and generate outline structures |
| Grammarly | check grammar and spelling (NO declaration needed) |
| GitHub Copilot | assist with code development for the data analysis pipeline |
| DeepL | translate Chinese draft sections into English |

## Full examples

### Example: Language editing with one tool

```text
Declaration of generative AI and AI-assisted technologies in the
manuscript preparation process

During the preparation of this work the authors used Claude (Anthropic)
in order to improve language clarity and readability. After using this
tool, the authors reviewed and edited the content as needed and take
full responsibility for the content of the published article.
```

### Example: Multiple tools

```text
Declaration of generative AI and AI-assisted technologies in the
manuscript preparation process

During the preparation of this work the authors used Claude (Anthropic)
in order to improve language clarity and readability, and ChatGPT
(OpenAI, GPT-4) in order to assist with organizing literature review
notes. After using these tools, the authors reviewed and edited the
content as needed and take full responsibility for the content of the
published article.
```

### Example: Translation assistance

```text
Declaration of generative AI and AI-assisted technologies in the
manuscript preparation process

During the preparation of this work the authors used DeepL and Claude
(Anthropic) in order to translate portions of the manuscript from Chinese
to English and to improve the readability of the English text. After
using these tools, the authors reviewed and edited the content as needed
and take full responsibility for the content of the published article.
```

## When NO declaration is needed

If the only AI-related tools used were:
- Basic grammar/spell checkers (Grammarly for grammar only)
- Reference managers (Zotero, Mendeley, EndNote)
- Standard search engines

Then no declaration section is needed. Do not add an empty declaration.

## Placement

- End of manuscript
- New section with the standard title
- Before the References section
- Part of the main manuscript file (not separate)
