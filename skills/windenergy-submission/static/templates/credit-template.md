# CRediT Author Contributions Template

## Author-first format (recommended)

Replace [Author Name] with actual names. Remove unused roles.

```text
Author contributions

[Author A]: Conceptualization, Methodology, Software, Writing - original
draft.
[Author B]: Data curation, Formal analysis, Validation.
[Author C]: Supervision, Funding acquisition, Writing - review & editing.
[Author D]: Resources, Investigation.
[Author E]: Visualization, Writing - review & editing.
```

## Role-first format (alternative)

```text
Author contributions

Conceptualization: [Author A], [Author C].

```

## Role descriptions (for reference when collecting from co-authors)

| Role | What it means |
|------|--------------|
| Conceptualization | Ideas; research goals and aims |
| Data curation | Data management, cleaning, organizing |
| Formal analysis | Statistical, mathematical, computational analysis |
| Funding acquisition | Getting financial support |
| Investigation | Doing the experiments, running the models |
| Methodology | Designing the methods |
| Project administration | Coordination and management |
| Resources | Providing data, equipment, access |
| Software | Writing code, building tools |
| Supervision | Oversight and mentorship |
| Validation | Checking results are correct |
| Visualization | Making figures and tables |
| Writing - original draft | Writing the first version |
| Writing - review & editing | Revising and polishing |

## Chinese collection template

Use this to collect contributions from Chinese co-authors:

```text
请各位作者提供以下信息：

1. 姓名（英文，与论文一致）
2. 主要贡献（可多选）：
   - 提出研究思路 (Conceptualization)
   - 数据处理和管理 (Data curation)
   - 数学/统计分析 (Formal analysis)
   - 获得经费 (Funding acquisition)
   - 做实验/跑模型 (Investigation)
   - 设计方法 (Methodology)
   - 项目管理 (Project administration)
   - 提供数据/资源 (Resources)
   - 写代码 (Software)
   - 指导研究 (Supervision)
   - 验证结果 (Validation)
   - 画图/做表 (Visualization)
   - 写初稿 (Writing - original draft)
   - 修改论文 (Writing - review & editing)
```

## Placement

Place this section:
- After Acknowledgements
- Before References
