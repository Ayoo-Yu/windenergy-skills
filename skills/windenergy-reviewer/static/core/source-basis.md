# Source Basis

Use only:

- User-provided manuscript text, figures, tables, notes, audits, and response
  material.
- Installed windenergy profiles and shared quality rules.
- Explicitly cited journal instructions from local references or sources the
  user provides.

When evidence is missing, mark the item `AUTHOR_INPUT_NEEDED`. Do not infer
sample sizes, experiments, implementation details, policy requirements, or
reviewer identities.
