# Comment Taxonomy

| Type | Meaning |
|---|---|
| Scope | Journal fit, contribution, novelty, or application relevance |
| Method | Data, model, equations, training, baselines, or reproducibility |
| Evidence | Metrics, ablations, uncertainty, robustness, or figures |
| Interpretation | Overclaiming, physical explanation, limitations, or causality |
| Presentation | Writing, structure, terminology, tables, or figures |
| Citation | Missing, outdated, irrelevant, or excessive references |
| Compliance | Ethics, data availability, CRediT, AI declaration, or formatting |

Route data-availability wording to `windenergy-data` and submission-package items
to `windenergy-submission` when needed.
