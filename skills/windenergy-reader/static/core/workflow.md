# Workflow

1. Identify source format and load only the matching source fragment.
2. Build a full-document source map before translating.
3. Create stable text, caption, figure, and table IDs.
4. Translate substantive blocks conservatively.
5. Crop figures and tables when source files permit.
6. Write `paper.md`, `source_map.json`, `translation_notes.md`, and `assets/`.
7. Answer follow-up questions from source IDs rather than memory.
