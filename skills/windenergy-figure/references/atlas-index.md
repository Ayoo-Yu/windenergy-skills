# Windenergy Figure Atlas

The atlas assets in `assets/chart-atlas/` are generated by
`assets/figures4papers/windenergy_atlas/generate_windenergy_atlas.py` from
synthetic demonstration data.

Included patterns:

1. Wind rose direction-speed distribution.
2. Turbine power curve with operating regions.
3. Forecast versus actual time series.
4. Error distribution by forecast horizon.
5. Prediction interval reliability.
6. Model comparison with confidence intervals.
7. Ablation heatmap.
8. Wake/layout farm schematic.
9. Storage and market optimization profile.
10. Mechanism workflow diagram.

Use these as pattern references, not as manuscript evidence. Replace synthetic
data with user-provided data before using a figure in a paper.
