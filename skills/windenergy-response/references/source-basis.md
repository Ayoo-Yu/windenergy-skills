# Source Basis

Use this file to keep response letters grounded in target-journal instructions
and manuscript facts.

## Source Hierarchy

1. The editor decision letter and reviewer comments.
2. Current target-journal revision instructions.
3. The revised manuscript and tracked changes.
4. Author-provided evidence, analyses, figures, and constraints.

## Rules

- A point-by-point response is the default structure.
- Each concern must be answered, revised, justified, or flagged as unresolved.
- Preserve reviewer wording unless the user asks for summarization.
- A cover letter is adjacent submission material; route full cover-letter drafting
  to `windenergy-submission`.
- Do not overstate what was changed or what new evidence proves.
